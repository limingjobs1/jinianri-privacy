使用指南

  • 添加纪念日：首页右下角 + → 填写标题、日期、分类，可开启本地提醒
  • 查看日历：底部「日历」Tab → 圆点表示当天有纪念日，点击日期查看详情
  • 记录瞬间：底部「瞬间」Tab → 撰写日记，可关联纪念日
  • 数据备份：「我的 → 导出 JSON 备份」导出；「从 JSON 恢复」导入
  • 清除数据：「我的 → 清除全部数据」，操作不可恢复，请先备份

常见问题

Q：数据存在哪里？
A：全部保存在您的 iPhone / iPad 本地，不会上传至任何服务器。

Q：换机如何迁移？
A：在旧设备导出 JSON 备份，通过 AirDrop / 邮件发送到新设备，在新设备上使用「从 JSON 恢复」。

Q：通知没有收到？
A：请确认 iOS「设置 → 通知 → 纪念日」已允许通知；应用内「我的 → 允许纪念日提醒」已开启；编辑纪念日时已开启「本地提醒」。

Q：农历显示不正确？
A：农历由 iOS 系统日历换算，请以系统显示为准。如有异常请提供截图与系统版本联系我们。

Q：如何删除单条纪念日？
A：点击列表项进入详情 → 底部「删除纪念日」；或长按列表项选择删除。
