纪念日 应用支持

「纪念日」是一款离线倒计时与日记工具，帮助您记录重要日期、查看农历、设置本地提醒并撰写瞬间记录。

主要功能

  • 纪念日倒计时与分类管理
  • 公历 / 农历日期显示
  • 月历浏览与搜索筛选
  • 本地通知提醒
  • 瞬间日记与 JSON 备份

常见问题

数据存在哪里？ 全部保存在您的 iPhone / iPad 本地，不上传云端。

如何备份？ 在应用「我的 → 导出 JSON 备份」中导出，需要恢复时使用「从 JSON 恢复」。

通知不生效？ 请在 iOS「设置 → 通知 → 纪念日」中确认已允许通知，并在应用内开启提醒。
