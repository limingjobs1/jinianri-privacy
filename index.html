纪念日 隐私政策

最后更新：2026年6月26日

1. 适用范围

本政策适用于「纪念日」iOS 应用。使用本应用即表示您同意本政策的条款。

2. 我们收集的信息

本应用不收集、不上传、不出售或共享您的个人信息。您添加的纪念日、瞬间日记、备注、提醒设置等数据均仅保存在您的设备本地，不会传输至任何服务器。

3. 权限说明

通知权限：仅在您主动开启提醒时使用，用于在纪念日临近时发送本地通知。通知内容仅来自您自行录入的信息。

文件访问：仅在您主动选择「从 JSON 恢复」时读取您选中的备份文件，不会后台访问其他文件。

本应用不访问相册、相机、通讯录、定位信息，也不进行网络数据传输。

4. 第三方服务

本应用不包含第三方广告、统计 SDK 或账号登录服务，不会将您的数据发送给第三方。

5. 数据导出与删除

您可在应用内「我的 → 数据管理」中导出 JSON 备份、导入备份，或一键清除全部本地数据。卸载应用后，所有本地数据将被 iOS 系统删除。

6. 儿童隐私

本应用不面向 13 岁以下儿童主动收集信息，也不会要求儿童提供个人数据。

7. 联系我们

如有隐私相关问题，请发送邮件至：limingjobs1@126.com

或通过 App Store 应用详情页的「App 支持」联系我们。
